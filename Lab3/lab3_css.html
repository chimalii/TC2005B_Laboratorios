Aplicación de estilos con CSS

1. Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

A pesar de que !important prioriza estilos, usarlo en exceso puede dificultar la mantenibilidad y la depuración del código, ya que puede anular estilos que querías que se aplicaran por razones específicas.
Así que es recomendable apegarse a especificidad y buena organización en el archivo CSS y usar esta instrucción sólo en casos realmente necesarios.

2. Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Si quieres colocar una imagen de fondo debes considerar su tamaño, tomando en cuenta los dispositivos en que se desplegará, para que se vea nítida y no pixeleada.

3. Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Si esperas diseños responsivos debes utilizar las unidades relativas(%) que permiten definir el ancho de un elemento en relación con el ancho de su contenedor.
Sin embargo, si deseas un control preciso sobre el tamaño de un elemento, indícalo con unidades absolutas:
* px (pixeles) es recomendable para dispositivos de pantalla con alta densidad de píxeles.
* pt (puntos) se utiliza comúnmente para definir el tamaño de fuentes en documentos destinados a impresión.

4. ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

El uso de una versión minimizada del CSS mejora el rendimiento del sitio porque reduce el tiempo de carga de la página, ya que el archivo es más ligero y el navegador lo procesa más rápido.
